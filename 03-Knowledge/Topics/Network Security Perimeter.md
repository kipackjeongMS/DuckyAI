## Summary

Network Security Perimeter (NSP) is an Azure networking feature that establishes logical network boundaries for PaaS resources, controlling inbound and outbound access at the service level without requiring VNet injection. AppConfig team is actively driving NSP adoption across services with Q2 delivery milestones confirmed.
